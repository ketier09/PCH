## Mejora de Seguridad

### Descripción

Se sugiere mejorar la seguridad del código moviendo las claves sensibles fuera del código. Esto se puede lograr utilizando variables de entorno para almacenar información sensible como claves API, contraseñas, etc.

### Tareas
- Identificar las claves sensibles en el código.
- Reemplazar las claves sensibles por variables de entorno.
- Documentar cómo configurar las variables de entorno en el proyecto.
