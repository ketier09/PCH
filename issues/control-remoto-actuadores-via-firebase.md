## Implementar Control Remoto de Actuadores vía Firebase Stream

### Descripción
Se requiere implementar un sistema que permita el control remoto de actuadores a través de Firebase stream. Esto facilitará la gestión y supervisión de dispositivos desde cualquier ubicación.

### Tareas
- [ ] Configurar Firebase en el proyecto.
- [ ] Implementar la lógica para suscribirse a los cambios en Firebase.
- [ ] Desarrollar la interfaz para el control de actuadores.
- [ ] Realizar pruebas de funcionamiento.

### Consideraciones
- Asegurarse de que la seguridad de los datos sea una prioridad.
- Documentar el proceso para futuras referencias.
